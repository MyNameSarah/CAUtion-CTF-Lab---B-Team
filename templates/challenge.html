Internal Share System

Employees use the internal share system to exchange private notes.

Shared objects are referenced using encoded identifiers.

Example Shared Link

/share/Z3Vlc3Rfbm90ZQ==

Available API

/api/share/check?key=VALUE

The API may help determine whether a shared object exists.

Objective

Discover hidden internal shared objects.

Shared object names often follow predictable naming conventions.
